Tentang Game

Kuis Battle adalah game kuis interaktif web oleh Kelompok 4. Berjalan di browser tanpa instalasi apapun. Cocok untuk HP, tablet, laptop, hingga TV Android & proyektor.

🛠️

Teknologi

  • HTML5 Canvas — animasi arena real-time
  • CSS3 — layout responsif, animasi
  • Vanilla JS — logika game, routing
  • Web Audio API — efek suara synthesizer
  • Fullscreen API — layar penuh untuk TV/proyektor
⚔️

Mode 1 — Tarik Tambang

  • 2 pemain, split screen kiri & kanan
  • 70 detik untuk jawab 20 soal
  • Benar: -3 detik waktu, skor +1, karakter maju
  • Salah: -5 detik waktu, soal masuk antrian lagi
  • Menang: selesai lebih dulu atau waktu lawan habis
🤝

Mode 2 — Tim Battle

  • 2 tim max 20 orang, berlomba bersamaan
  • Soal bergilir per anggota, 70 detik per giliran
  • Tim selesai 20 soal lebih dulu = MENANG!
🌿

Mode 3 — Cozy Mode

  • Solo player, tanpa timer
  • Jawab → lihat hasil + penjelasan → tekan Lanjut
  • Salah ditunjukkan jawaban benar + penjelasan
  • Karakter berjalan ke finish setiap soal
💡

Fitur Khusus

  • 🎵 Musik latar otomatis mengecil saat efek suara
  • ▶ Resume game yang belum selesai dari home
  • ⛶ Fullscreen — ideal untuk TV Android & proyektor
  • ⏸ Pause/Resume/Restart dari dalam game
  • 📋 Lihat semua soal + penjelasan setelah selesai
