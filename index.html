🪜 Cara Pembuatan

1
Riset & Perencanaan

Menentukan konsep game, mode permainan, dan soal-soal pengetahuan umum Indonesia yang relevan dan edukatif.

2
Desain UI/UX

Merancang wireframe dan tampilan visual menggunakan Figma, memilih palet warna merah-putih sesuai identitas Indonesia.

3
Pengembangan Arena

Membuat animasi arena tarik tambang menggunakan Canvas API. Karakter dirender dengan JavaScript murni tanpa library eksternal.

4
Logika Game

Mengimplementasikan sistem timer, skor, antrian soal, mode 1v1, tim, dan cozy menggunakan state machine sederhana.

5
Efek Suara

Membuat efek suara menggunakan Web Audio API (synthesizer) tanpa file audio tambahan, kecuali musik latar.

6
Testing & Optimasi

Menguji di berbagai perangkat (HP, tablet, laptop, TV Android), mengoptimasi layout responsif dan performa canvas.

📖 Tentang Game

Kuis Battle — Game kuis interaktif berbasis web

🎮

Kuis Battle

Game kuis interaktif berbasis web yang dibuat oleh Kelompok 4. Berjalan sepenuhnya di browser tanpa instalasi.

⚔️ Mode 1 — Tarik Tambang

  • 2 pemain, split screen kiri & kanan
  • 5 menit total untuk menjawab 20 soal
  • Benar: skor +1  |  Salah: -15 detik dari total waktu
  • Selesai 20 soal lebih dulu atau waktu lawan habis = MENANG

🤝 Mode 2 — Tim Battle

  • 2 tim max 20 orang, berlomba serentak
  • 5 menit total per tim untuk 20 soal
  • Benar: skor +1  |  Salah: -15 detik dari sisa waktu tim
  • Tim selesai 20 soal lebih dulu = MENANG

🌿 Mode 3 — Cozy Mode

  • Solo player, tanpa timer
  • Jawab → hasil + penjelasan → Lanjut
  • Karakter berjalan ke garis finish
  • Selesai 20 soal = sampai FINISH!

💡 Fitur

  • 🎵 Musik latar otomatis duck saat efek suara
  • ▶ Resume game yang belum selesai
  • ⛶ Fullscreen untuk TV Android & proyektor
  • ⏸ Pause / Resume / Restart dari game
  • 📱 Responsif di semua ukuran layar
